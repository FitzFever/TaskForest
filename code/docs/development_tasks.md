## 开发环境配置

### 必要软件
- Node.js 18+
- pnpm
- Visual Studio Code
- Git
- SQLite

### 推荐 VS Code 插件
- ESLint
- Prettier
- TypeScript and JavaScript
- Three.js Editor
- SQLite Viewer
- Git Lens

### 开发命令
```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 运行测试
pnpm test

# 构建应用
pnpm build

# 打包应用
pnpm package
```
